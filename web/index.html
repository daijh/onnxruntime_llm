Web Demos

Browser-based LLM demos running entirely on-device with WebGPU — no server-side inference.

LLM Runner →
General-purpose LLM chat interface for any merged-decoder ONNX model. Hand-rolls the decode loop directly on ONNX Runtime Web; capabilities are auto-detected from the ONNX session metadata.
Transformers Runner →
Browser LLM chat interface powered by Transformers.js (AutoModelForCausalLM.generate()) on WebGPU — the full generation pipeline runs in the library, so no per-model configuration is required.
